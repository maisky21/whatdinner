# Blueprint: 오늘의 미식 운세 (Gourmet Fortune)

## 서비스 개요
"오늘의 미식 운세"는 사용자의 현재 상태나 선택한 카테고리에 맞춰 AI가 저녁 메뉴와 그에 따른 운세를 추천해주는 웹 서비스입니다. 인스타그램 포스트 스타일의 감성적인 디자인과 실시간 이미지 생성 기능을 제공합니다.

## 주요 기능 및 디자인 상세
- **메뉴 추천 시스템**: 한식, 양식, 일식, 중식, 간단식 카테고리별 데이터셋을 바탕으로 랜덤 추천.
- **인스타그램 스타일 UI**: 친숙하고 세련된 SNS 포스트 형태의 결과 화면 제공.
- **실시간 이미지 생성**: `html2canvas`를 사용하여 결과 화면을 고해상도 이미지로 변환.
- **보내는 사람 설정**: 추천 결과에 '아빠가', '여보가' 등 커스텀 닉네임 포함 기능.
- **랜덤 닉네임 버튼 (🎲)**: 클릭 시 재미있는 닉네임을 자동으로 설정.
- **모바일 최적화 저장**: 이미지 생성 후 모달 팝업을 통해 롱프레스(길게 누르기)로 저장하도록 안내.
- **다국어 지원**: 한국어와 영어를 지원하며 실시간 언어 전환 가능.
- **다크/라이트 모드**: 사용자의 OS 설정 반영 및 수동 전환 지원.

## 현재 진행된 작업 및 변경 사항
1. **모바일 레이아웃 최적화 (핵심)**:
   - `@media (max-width: 600px)` 환경에서 결과 카드의 이미지 크기를 20% 축소하고 전체적인 상하 여백을 컴팩트하게 조정.
   - 결과 카드가 화면 높이의 80%를 넘지 않도록 설정하여 하단 버튼까지 한 화면에 들어오도록 최적화.
2. **이미지 저장 로직 완성 및 팝업 안내**:
   - 저장 버튼 클릭 시 다운로드 대신 이미지 모달 팝업이 뜨도록 로직 확정.
   - 안내 문구를 "이미지를 꾹 눌러 저장하세요! 📸"로 간소화하여 가독성 향상.
3. **UI 마감 및 가독성 향상 (섹션 구분 & 하이라이트)**:
   - **섹션 구조화**: 메뉴 이름, 조리법, 영양 성분, 미식 운세 섹션 사이에 연한 구분선(`border-bottom`)과 상하 여백을 추가하여 가독성 극대화.
   - **이모지 활용**: 각 섹션 제목 앞에 관련 이모지(🍳, 🌿, ✨)를 추가하여 시각적 직관성 제공.
   - **폰트 및 강조**: 본문 `line-height`를 1.6으로 최적화하고, 주요 키워드(행운, 성공, 에너지 등)에 하이라이트 효과(배경 그라데이션) 적용.
   - **데이터 확충**: 전 메뉴(25종)에 대해 조리법(`recipe`) 및 영양 성분(`nutrition`) 데이터 추가 완료 (국문/영문).
4. **ads.txt 및 Cloudflare 리다이렉션 오류 해결**:
   - **ads.txt 추가**: 루트 폴더에 Adsense 인증을 위한 `ads.txt` 파일 생성 및 데이터 입력.
   - **Cloudflare 설정 보완**: `_redirects` 파일을 생성하여 `/ads.txt` 경로가 `index.html` 등으로 리다이렉트되지 않고 200 상태 코드로 직접 서비스되도록 강제 설정.
5. **Google Analytics 4 (GA4) 연동**:
   - 모든 서비스 페이지(`index.html`, `about.html`, `privacy.html`, `terms.html`)의 `<head>` 태그 최상단에 GA4 추적 코드(ID: G-0DTR26JE6T)를 삽입하여 방문자 분석 환경 구축 완료.

## 향후 계획
- 더 다양한 메뉴 데이터 확충.
- 공유하기 기능(Web Share API) 연동 검토.
- 방문자 통계 분석을 위한 태그 추가.
